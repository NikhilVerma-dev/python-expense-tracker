# Expense Tracker

A simple Python project that helps students track their daily expenses.

## Features

- Add expenses
- View expense reports
- Calculate total expenses
- Object-oriented design using Python classes

## Technologies Used

- Python
- Classes & Objects
- Lists
- Loops
- Conditional Statements

## Future Improvements

- Categories (Food, Travel, Shopping)
- CSV file storage
- Monthly reports
- Data visualization using Matplotlib

## How to Run

```bash
Student Expense Tracker.py
```
